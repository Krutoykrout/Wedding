# Денис & Лидия — свадебное приглашение

Готовая адаптивная версия сайта для GitHub Pages.

## Как загрузить

1. Распакуйте архив.
2. Загрузите в корень репозитория все файлы и папки из архива: `index.html`, `style.css`, `script.js`, `.nojekyll`, папку `assets`.
3. В настройках GitHub Pages выберите публикацию из ветки `main` и папки `/root`.

## Что внутри

- адаптивная версия для ПК и телефонов;
- анимированный таймер;
- плавные появления секций;
- анимация письма и жемчужины;
- анимация катящейся жемчужины по склону без подписи;
- музыка с попыткой автозапуска;
- форма подтверждения присутствия.

Важно: некоторые браузеры могут блокировать звук до первого касания, клика или прокрутки. Это ограничение браузеров, не GitHub Pages.


## Последние правки

- из блока дресс-кода убрана формулировка про цветные/светлые элементы;
