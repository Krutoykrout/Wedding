# Денис & Лидия — свадебное приглашение

Основано на исходном шаблоне wedding-invite-site.

## Что внутри
- `index.html` — страница приглашения.
- `styles.css` — стили и анимации.
- `script.js` — анимации, запуск музыки, таймер, RSVP.
- `assets/music.mp3` — выбранная песня.
- `assets/couple.jpg` — фото молодожёнов, используется на сайте один раз.
- `assets/dress-men.jpg`, `assets/dress-women.jpg` — фото для дресс-кода.

## RSVP
Форма отправляет заявки на `ekenotova@yandex.ru` через FormSubmit без открытия почты гостя. При первой заявке FormSubmit может попросить подтвердить адрес получателя.
