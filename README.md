# Приглашение Дениса и Лидии

Статичный сайт-приглашение по мобильному шаблону: стартовый экран-письмо, анимации, тайминг, дресс-код, RSVP-форма.

## RSVP

Форма отправляет ответы на: `ekenotova@yandex.ru` через FormSubmit.

Важно: при первой отправке FormSubmit обычно присылает письмо-подтверждение на `ekenotova@yandex.ru`. Нужно один раз открыть это письмо и активировать адрес, после этого заявки будут приходить автоматически. Почта гостя не используется, почтовый клиент гостя не открывается.

## Музыка

Предыдущий аудиофайл не включён в эту версию. Когда будет новый трек, поместите его в папку:

`assets/music.mp3`

и в `index.html` у тега `<audio id="weddingMusic">` укажите:

`src="assets/music.mp3"`

После этого музыка будет запускаться при нажатии «Открыть приглашение».

## Фото

Фото пары используется на сайте только один раз — в первом основном экране после открытия приглашения.
