# Лидия & Денис — свадебное приглашение

Мобильный сайт-приглашение с начальным экраном-письмом, музыкой после нажатия «Открыть приглашение», анимациями при скролле, фотографиями для дресс-кода и одной главной фотографией молодожёнов.

## Что менять

- Тексты и даты: `index.html`
- Цвета и анимации: `styles.css`
- Музыка, таймер и RSVP: `script.js`
- Изображения и аудио: папка `assets/`

## Музыка

Файл подключён как `assets/music.mp3`. Браузеры запускают звук только после действия пользователя, поэтому музыка стартует после нажатия «Открыть приглашение».
